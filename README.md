# ColorPicker Component

This is a simple Color Picker React component that allows users to select a color and automatically adjusts the font color for better readability. Additionally, users can copy the selected color value to the clipboard.

## Features

- Color selection using a color input element.
- Automatic font color adjustment for better readability.
- Copy selected color value to the clipboard.

## Installation

1. Clone the repository:

```sh
git clone https://github.com/mayurr11/color-picker-app.git
cd color-picker-app
```

2. Install the dependencies:

```sh
npm install
```

## Usage

1. Import and use the `ColorPicker` component in your React application.

2. Ensure you have `tinycolor2` installed for color manipulation:

```sh
npm install tinycolor2
```
